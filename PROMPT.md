# Промпт для создания лендинг-страницы Yora (Светлая тема)

## Общее описание страницы

Создай современную лендинг-страницу в светлой теме с градиентным фоном, видео-фоном и анимированным Aurora эффектом. Страница должна быть полностью адаптивной, без скролла (все помещается на экран), с центрированным контентом и тремя кнопками-ссылками внизу.

## Структура страницы

### 1. Фон и эффекты

**Основной фон:**
- Градиент: `bg-gradient-to-br from-gray-50 to-white` (от светло-серого к белому)
- Высота и ширина: `h-screen w-screen` (на весь экран)
- Overflow: `overflow-hidden` (без скролла)

**Видео фон:**
- Файл: `/yora-ai.webm` (находится в папке `public/`)
- Позиционирование: абсолютное, на весь экран (`absolute inset-0`)
- Стили: `object-cover opacity-50` (покрывает весь экран, прозрачность 50%)
- Автовоспроизведение: `autoPlay loop muted playsInline`
- Overlay поверх видео: градиент `from-white/60 via-white/40 to-white/60` для читаемости текста

**Aurora эффект (WebGL):**
- Библиотека: `ogl` (WebGL)
- Цвета градиента: `['#82D5FF', '#8B8EFF', '#7E3FFC', '#AB57FF', '#D35EF8', '#FF66EF']`
- Прозрачность: `opacity-20` (20%)
- Параметры: `amplitude: 0.8`, `blend: 0.5`
- Позиционирование: абсолютное, на весь экран

## Технические детали

### Цветовая палитра
- Основной текст: `text-gray-900` (#111827)
- Вторичный текст: `text-gray-600` (#4B5563)
- Фон кнопок: `bg-white/90` (белый 90% прозрачности)
- Границы: `border-gray-200` (#E5E7EB)
- Hover стрелка: `#016fee` (синий)

### Шрифты
- Заголовок: Bold, размер 4xl (36px) на мобильных, 5xl (48px) на десктопе
- Описание: Regular, размер lg (18px)
- Кнопки: Semibold, размер base (16px)

### Адаптивность
- Контейнер: `max-w-md` (максимум 448px)
- Отступы: `px-4 py-4` (16px по горизонтали и вертикали)
- На мобильных: все элементы центрированы, кнопки на всю ширину

### Анимации и переходы
- Все переходы: `duration-300` (300ms)
- Hover эффекты: плавные, с `transition-all`
- Aurora эффект: непрерывная анимация через WebGL

## Важные замечания

1. **Без скролла:** Страница должна полностью помещаться на экран (`h-screen`, `overflow-hidden`)
2. **Видео:** Должно автоматически воспроизводиться, зацикливаться, быть без звука
3. **Центрирование:** Весь контент центрирован вертикально и горизонтально
4. **Z-index:** Контент должен быть поверх видео и Aurora эффекта (`z-10`)
5. **Адаптивность:** На мобильных устройствах все должно оставаться читаемым и функциональным
